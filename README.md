# ybigta_first_proj
YBIGTA 2019 여름방학 프로젝트

# 서버에 Git 연동 방법
- git clone https://github.com/etture/ybigta_first_proj.git
- cd ybigta_first_proj
- git checkout <자기 브랜치 이름>
===== 일단 설정 완료
- 파일 새로 작업하고 나서
- git add <바꾼 파일들>
- git commit -m "커밋 메시지, 그냥 뭐 작업했는지 짤막하게 기록하면 됨"
- git push origin <자기 브랜치 이름>
